# 网感训练板

一个纯前端本地工具，帮你系统化训练"网感"——拆解爆款内容、预测传播潜力、验证判断。

## 功能

- **📝 添加内容** — 手动录入或粘贴链接解析，20+ 中国主流平台支持
- **🔍 拆解分析** — 钩子强度、情绪触发、可复用结构、评论区洞察
- **🎮 训练中心** — 三种模式：拆解填空 / 维度聚焦 / 评论预言
- **📊 统计面板** — 各平台雷达图、准确率追踪、历史记录

## 技术

纯静态 HTML，本地存储，无需后端。AI 解析需要自行配置 API Key（在更多 → API 设置）。
