钩子层

□ 前3秒/前5字有没有让人停下来的理由？

□ 有没有反常识、悬念、冲突、利益承诺？

□ 如果我是用户，我会点进去吗？


情绪层

□ 引发了什么情绪？（愤怒？焦虑？爽感？共鸣？好奇？）

□ 情绪强度够不够？有没有代入感？

□ 评论区的人在吵什么？为什么吵？


结构层

□ 信息密度是否合适？有没有水内容？

□ 节奏如何？有没有拖沓或跳跃？

□ 结尾有没有引导互动/转发/关注的钩子？


可复用层

□ 这个结构/钩子能迁移到我的内容吗？

□ 换个领域/人群，这个选题还成立吗？

📖 网感训练板是什么？

这是一个帮你系统训练网感直觉的本地工具——也就是训练你对爆款内容为什么火的敏感度。

🚀 快速上手（3 步走）

① 添加内容 — 把你刷到的爆款标题粘贴进来，给它打分、拆解为什么火

② 训练中心 — 用案例库里的内容考自己：钩子是什么？情绪怎么触发的？杀手锏维度是哪个？

③ 查看统计 — 看你在哪些平台直觉最准，哪些维度最弱

📚 四个模块

➕ 添加内容
采集爆款案例，拆解钩子/情绪/结构/评论洞察，给综合评分
📚 案例库
管理所有采集的案例，支持按平台/时间筛选、搜索
🎮 训练中心
三种模式考自己：拆解填空、维度聚焦、评论预言
📊 统计
今日训练数、准确率、各平台网感雷达图

🤖 AI 辅助（可选）

在「设置 → API 设置」里接入 OpenAI / Kimi / 自定义 API，可以：自动解析内容、提炼标签、补全传播驱动力

💡 小技巧

  • 所有数据保存在浏览器本地，不上传服务器
  • 「设置 → 导出数据」可以备份 JSON
  • 拖拽「采集到剪贴板」书签到书签栏，一键采集网页内容
